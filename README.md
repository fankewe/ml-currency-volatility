# ml-currency-volatility
 Machine learning for FX volatility (EUR/USD)
## Machine Learning for Currency Volatility

This project uses high-frequency EUR/USD data to construct and model volatility.

- Data: EUR/USD 15-min (2023–2025)
- Method: Log returns + rolling volatility
- Tools: Python, Pandas, Matplotlib

🔗 Open notebook: demo.ipynb
